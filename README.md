//如何解决上述问题：1.识别：全面列举情绪词汇，让人做选择题，而非简答题；2.表达/宣泄：写情绪日记这件事本身就是一种宣泄情绪的方法；3.认识：认识情绪的本质，识别其来源，最后，从情绪中挣脱，趋向于解决问题
